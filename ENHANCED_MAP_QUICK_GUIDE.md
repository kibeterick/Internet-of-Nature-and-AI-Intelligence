# 🗺️ Enhanced Map & Error Monitoring - Quick Guide

## 🎮 Quick Start

### Enhanced Map

```
1. Click "Enhanced Map" tab
2. See FPS counter in top-right
3. Toggle layers in sidebar
4. Click zoom +/- buttons
5. Hover over nodes for info
6. Click nodes for details
```

### Error Monitor

```
1. Click "Error Monitor" tab
2. View error statistics
3. Filter by severity
4. Click "Resolve" to fix
5. Click "Clear All" to reset
```

---

## 🎛️ Map Controls

### Layers

- **Sensors**: Toggle sensor data
- **Vegetation**: Toggle vegetation layer
- **Heatmap**: Toggle heat distribution
- **Satellite**: Toggle satellite view
- **Weather**: Toggle weather data

### Zoom

- **+**: Zoom in (max 3x)
- **−**: Zoom out (min 0.5x)

### Interactions

- **Hover**: See node tooltip
- **Click**: Select node
- **Drag**: Pan map (ready)

---

## 📊 Error Severity Levels

| Level    | Color  | Icon | Meaning         |
| -------- | ------ | ---- | --------------- |
| Low      | Blue   | ℹ️   | Minor issue     |
| Medium   | Yellow | ⚠️   | Moderate issue  |
| High     | Orange | ⚠️   | Serious issue   |
| Critical | Red    | ❌   | System breaking |

---

## 🌤️ Weather Data

When weather layer is enabled:

- **Temperature**: Current temp in °C
- **Humidity**: Moisture level %
- **Wind Speed**: km/h
- **Condition**: Weather description

Updates every 3 seconds.

## 🔧 Troubleshooting

### Map Slow

- Disable unused layers
- Zoom to smaller area
- Check FPS counter
- Clear browser cache

### Errors Not Showing

- Click "Refresh" button
- Check filter setting
- Clear browser cache
- Reload page

### Weather Not Updating

- Enable weather layer
- Wait 3 seconds
- Check sidebar
- Refresh page

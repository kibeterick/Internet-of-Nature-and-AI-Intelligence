**Steps to Complete:**

2. Double-click it
3. Press any key when prompted
4. Wait 30-45 minutes
5. APK will be at: `mobile_app\build\app\outputs\flutter-apk\app-debug.apk`

**Option 2: Command Line**


**What Happens:**

1. Deletes corrupted NDK folder (1 min)
2. Cleans build files (1 min)
3. Downloads fresh Android NDK - 800 MB (15-20 min)
4. Downloads Flutter dependencies (5 min)
5. Compiles Dart to native Android (10-15 min)
6. Creates APK file (2 min)

**Requirements:**

- ✅ Stable internet connection (for 800 MB download)
- ✅ 2 GB free disk space
- ✅ Don't close window during build
- ✅ Don't interrupt the process

## 🎮 How to Use

### Access Ecosystem Health

```
1. Click "Ecosystem Health" in navigation
2. Select a monitoring area
3. View health metrics (Air, Water, Soil, Biodiversity)
4. Check active alerts
5. Click "Analyze Area" for AI insights
6. Interact with the map (toggle layers, click nodes)
```

### Access AI Model Tuning

```
1. Click "AI Tuning" in navigation
2. View current model performance metrics
3. Click "Start Fine-tuning" to optimize
4. Click "Test Prediction" to validate
5. Click "Add Training Data" for more examples
6. Review recommendations for improvement
```

## 🔧 Troubleshooting

### Issue: Error Still Showing

**Solution**:

1. Press Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
2. Close and reopen browser
3. Check browser console for errors

### Issue: Features Not Visible

**Solution**:

1. Verify you're logged in
2. Check if correct tab is selected
3. Scroll down if content is below fold
4. Check browser console for errors

### Issue: Slow Performance

**Solution**:

1. Close other browser tabs
2. Clear browser cache
3. Disable browser extensions
4. Check internet connection

### Issue: AI Features Not Working

**Solution**:

1. Verify internet connection
2. Check if API key is configured
3. Check browser console for errors
4. Try again in a few moments

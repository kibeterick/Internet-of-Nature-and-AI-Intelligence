### Essential Commands

```bash
# Navigate to app
cd "C:\Users\HP\Internate of Nature and AI intelligence\mobile_app"

# Install dependencies
C:\flutter\bin\flutter.bat pub get

# Configure Firebase
flutterfire configure

# Run app
C:\flutter\bin\flutter.bat run

# Build APK
C:\flutter\bin\flutter.bat build apk --release
```

## 📞 Still Need Help?

### Check these in order:

1. START_HERE_MOBILE_APP.md (troubleshooting section)
2. mobile_app/FLUTTER_APP_GUIDE.md (troubleshooting section)
3. Run: `C:\flutter\bin\flutter.bat doctor`

### Common Issues:

- **Flutter not found**: Use full path `C:\flutter\bin\flutter.bat`
- **Firebase error**: Run `flutterfire configure`
- **Build error**: Run `flutter clean` then `flutter pub get`
- **No devices**: Connect phone or start emulator

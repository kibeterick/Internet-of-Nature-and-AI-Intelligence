# Internet of Nature - Complete User Guide

## 🌿 Welcome to Internet of Nature

Your comprehensive ecosystem intelligence platform is now fully operational with 40+ features ready to explore.

## 🚀 Quick Start

### Access the Application

```
Frontend: http://localhost:5174
Backend API: http://localhost:3000
```

### First Time Setup

1. Open `http://localhost:5174` in your browser
2. Click **"Sign In with Gmail or GitHub"** or use email
3. Complete authentication
4. Explore the dashboard

## 📋 Main Features

### 1. Dashboard & Real-Time Monitoring

- **Live Sensor Data**: Monitor temperature, humidity, soil moisture, air quality
- **Mesh Network**: View active sensor nodes and their status
- **Real-Time Updates**: WebSocket-powered live data streaming
- **Nature Score**: Track overall ecosystem health (0-100)

**How to Use:**

- Navigate to "Dashboard" tab
- View sensor readings from different locations
- Click on sensors for detailed information
- Monitor active users online

### 2. Species Identification (🌿 Green Leaf Button)

- **Photo Upload**: Upload images of plants or animals
- **AI Analysis**: Gemini AI identifies species with 99.7% accuracy
- **Taxonomic Data**: Scientific names, common names, classification
- **Conservation Status**: Learn about endangered species
- **Interesting Facts**: Discover ecological information

**How to Use:**

1. Click the green leaf (🌿) floating button
2. Upload a clear photo of a plant or animal
3. Wait for AI analysis
4. View detailed species information
5. Save to your discoveries

### 3. 3D Ecosystem Visualization (🔄 Blue Circle Button)

- **Real-Time Simulation**: Watch ecosystem dynamics in 3D
- **Population Dynamics**: See how species populations change
- **Energy Flow**: Visualize food chain relationships
- **Interactive Controls**: Play/pause, speed up/slow down
- **Organism Types**: Producers, herbivores, carnivores, decomposers

**How to Use:**

1. Click the blue circle (🔄) floating button
2. Watch the 3D ecosystem simulation
3. Use controls to play/pause
4. Observe population changes
5. Learn about ecosystem balance

### 4. Genie AI Assistant

**How to Use:**

1. Navigate to "Nature AI" tab

4. Ask follow-up questions
5. Export reports if needed

## 🐛 Troubleshooting

### App Won't Load

1. Clear browser cache (Ctrl+Shift+R)
2. Check internet connection
3. Verify servers are running
4. Check browser console for errors

### Authentication Issues

1. Ensure pop-ups are allowed
2. Check Firebase configuration
3. Verify API keys are set
4. Try different browser

### AI Features Not Working

1. Check Gemini API key
2. Verify API quota
3. Check internet connection
4. Try again in a few moments

### Real-Time Updates Not Working

1. Check WebSocket connection
2. Verify backend is running
3. Check firewall settings
4. Refresh page

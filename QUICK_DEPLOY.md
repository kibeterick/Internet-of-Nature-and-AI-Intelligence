# 🚀 Quick Deploy - Get Online in 2 Minutes!

## Fastest Way (Firebase - Already Configured!)

### Step 1: Install Firebase CLI (One-time setup)

```bash
npm install -g firebase-tools
```

### Step 2: Login to Firebase

```bash
firebase login
```

### Step 3: Deploy!

```bash
npm run deploy:firebase
```

**OR** just double-click: `deploy-to-firebase.bat`

---

## Your Live URLs

After deployment, your app will be available at:

- **Primary**: https://flutter-ai-playground-214d7.web.app
- **Alternative**: https://flutter-ai-playground-214d7.firebaseapp.com

## Update Your App

Made changes? Deploy updates instantly:

```bash
npm run deploy:firebase
```

Changes go live in ~30 seconds!

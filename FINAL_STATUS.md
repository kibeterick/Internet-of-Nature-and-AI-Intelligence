## 🆘 Troubleshooting

### Server won't start
```bash
# Kill existing processes
taskkill /F /IM node.exe
npm run dev
```

### Styling not working
```bash
# Rebuild with Tailwind
npm run dev
# Hard refresh browser (Ctrl+Shift+R)
```

### Build fails
```bash
npm run clean
npm install
npm run build
```

### Port already in use
Edit `.env`:
```env
PORT=3001
```
